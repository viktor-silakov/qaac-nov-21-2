## Home Tasks Kit, lecture-2

### Install
`npm install`

### Run tests
all tests:

`npm test`

specific file:

`npx wdio wdio.conf.js --spec <path to file>`

for example:

`npx wdio wdio.conf.js --spec specs/message-service.js`

### Generate allure
`npm run report`

### Clean allure
`npm run clean`

### Run REPL with Chrome
`npm run repl`
